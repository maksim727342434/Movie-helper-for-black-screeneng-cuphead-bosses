To install put all the files from the zip folder to the game folder c:/Program Files (X86)/Steam/steamapps/common/Cuphead
if you are on linux put them here ~/.steam/steam/steamapps/common/Cuphead/ but to make it actually work on linux right click on the game in steam click properties and inside of launch options type that WINEDLLOVERRIDES="winhttp.dll=n,b" %command%
to access press m on any moment but do not select any bosses on the title screen because it will wipe your save file
